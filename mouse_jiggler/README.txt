MOUSE JIGGLER
=============

This program keeps your computer active by moving the mouse slightly every 15 seconds.

WINDOWS USERS:
1. Make sure Python is installed (download from python.org if needed)
2. Double-click "run_mouse_jiggler.bat" 
3. Press Ctrl+C to stop

MAC/LINUX USERS:
1. Open Terminal
2. Run: python3 mouse_jiggler.py
3. Press Ctrl+C to stop

The script will automatically install the required Python package (pyautogui) on first run.

FILES INCLUDED:
- mouse_jiggler.py (the main script)
- run_mouse_jiggler.bat (Windows launcher)
- README.txt (this file)
